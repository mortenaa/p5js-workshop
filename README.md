# p5.js Workshop

This is a fast, static website to host a programming workshop for children to learn creative programming with p5.js.

## Tech Stack

*   **Astro:** Static Site Generator
*   **Tailwind CSS:** Styling
*   **GitHub Pages:** Hosting

## How to run locally

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Start the development server:**
    ```bash
    npm run dev
    ```
    The website will be available at `http://localhost:4321`.

## How to deploy to GitHub Pages

The website is automatically deployed to GitHub Pages when you push to the `main` branch.

The GitHub Actions workflow in `.github/workflows/deploy.yml` will handle the build and deployment process.
