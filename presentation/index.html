p5.js Workshop

En introduksjon til kreativ koding

Grunnleggende Struktur

Alle p5.js-skisser har to hovedfunksjoner: setup() og draw().


// Kjøres én gang når programmet starter
function setup() {
  // Oppsett-kode her, f.eks. createCanvas()
}

// Kjøres i en løkke (om og om igjen) etter setup()
function draw() {
  // Tegne-kode her
}

Variabler og Datatyper

Brukes til å lagre informasjon.


let score = 0; // Kan endres
const GRAVITY = 9.8; // Kan ikke endres

Bruk let for verdier som kan endres, og const for konstante verdier.

Datatyper

  • Tall (Number): let alder = 15;
  • Tekst (String): let navn = "Ola";
  • Array (Liste): let farger = ["rød", "grønn", "blå"];

Operatorer

  • Aritmetiske: +, -, *, /
  • Tilordning: =, +=, -=
  • Sammenligning: ===, <, >

Kontrollstrukturer

Betingelser (if / else)

Kjører kode kun hvis en betingelse er sann.


if (x > width) {
  // Gjør noe
} else if (x < 0) {
  // Gjør noe annet
} else {
  // Til slutt
}

for-løkker

Repeterer kode et bestemt antall ganger.


for (let i = 0; i < 5; i++) {
  // Denne koden kjøres 5 ganger
  ellipse(i * 50, 100, 40, 40);
}

Tegning i p5.js

Alt skjer på et "lerret" (canvas).

Koordinatsystemet

  • (0,0) er øverst i venstre hjørne.
  • X-verdien øker mot høyre.
  • Y-verdien øker nedover.
  • width og height er lerretets bredde og høyde.

Farger

  • createCanvas(bredde, høyde)
  • background(farge)
  • fill(farge), noFill()
  • stroke(farge), noStroke()

background(200); // Grå
fill(255, 0, 0); // Rød
stroke(0, 0, 255); // Blå

Former

  • ellipse(x, y, bredde, høyde)
  • rect(x, y, bredde, høyde)
  • triangle(x1, y1, x2, y2, x3, y3)
  • beginShape(), vertex(x, y), endShape(CLOSE)

Input fra Bruker

Gjør skissene interaktive.

Mus

  • mouseX, mouseY: Musens posisjon.
  • mouseIsPressed: Sjekker om museknapp er nede.
  • mouseReleased(): Funksjon som kalles ved klikk.

Tastatur

  • keyIsPressed: Sjekker om en tast er nede.
  • key: Holder verdien av siste tast.

if (keyIsPressed && key === 'a') {
  // Beveg spilleren til venstre
}

Bilder og Matematikk

Bilder

  • loadImage("fil.png"): Laster inn et bilde.
  • image(bilde, x, y, b, h): Viser bildet.

Matematikk

  • random(min, max): Tilfeldig tall.
  • sin(), cos(): For sirkulær bevegelse.
  • radians(grader): Konverterer grader til radianer.
